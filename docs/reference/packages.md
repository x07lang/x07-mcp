# Packages

`x07-mcp` publishes these MCP kit packages:

- `ext-mcp-core@0.2.2`: protocol constants, JSON-RPC helpers, diagnostics, SSE/event-id helpers
- `ext-mcp-toolkit@0.2.2`: server/tools manifest loaders, schema helpers, shared dispatcher, progress/cancel/subscriptions
- `ext-mcp-worker@0.2.2`: worker protocol + worker entrypoint
- `ext-mcp-sandbox@0.2.2`: router-side sandbox + worker spawn helpers (streaming/cancel-aware)
- `ext-mcp-transport-stdio@0.2.2`: stdio MCP transport (progress, cancellation, subscriptions)
- `ext-mcp-transport-http@0.2.0`: HTTP MCP transport + SSE helpers
- `ext-mcp-auth@0.1.0`: OAuth test-static validation + challenge/PRM helpers
- `ext-mcp-obs@0.1.0`: audit JSONL + metrics hooks
- `ext-mcp-rr@0.2.2`: deterministic stdio, HTTP, and HTTP+SSE replay helpers + sanitizers
